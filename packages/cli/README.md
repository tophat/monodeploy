# Monodeploy CLI

This package exposes a CLI for Monodeploy. It provides an alternative to the Node API.

## Installation

```sh
yarn add monodeploy
```

## Usage

```sh
yarn monodeploy --cwd <PATH_TO_MONOREPO> \
    --dry-run --log-level 0 \
    --conventional-changelog-config @tophat/conventional-changelog-config
```
